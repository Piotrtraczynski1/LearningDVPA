\englishtitle   {English title}
\polishabstract {
Tematem pracy licencjackiej jest implementacja oraz analiza wydajności algorytmu aktywnego uczenia się automatów visibly pushdown \cite{main}.
Przedstawiono motywację do aktywnego uczenia się automatów. Opisano algorytm wraz ze szczegółami implementacyjnymi oraz wykorzystanie dodatkowej
wskazówki usprawniającej jego działanie \cite{srs}. Następnie przedstawiono sposób generowania automatów oraz ich wykorzystanie do testowania algorytmu.
Na końcu przedstawione zostały wyniki eksperymentów.

}
\englishabstract{
The aim of this thesis is the implementation and performance analysis of an active learning algorithm for visibly pushdown automata \cite{main}.
The motivation for active automata learning is presented. The algorithm is described with its implementation details, as well as the use of additional
information which improves the algorithm's performance \cite{srs}. Then, the method of generating automata and their use in testing
the algorithm are presented. Finally, the results of the experiments are presented.

}
