\polishtitle    {Uczenie się DVPA z więzami}
\englishtitle   {Learning DVPA with constraints}
\polishabstract {
Tematem pracy licencjackiej jest implementacja oraz analiza wydajności algorytmu aktywnego uczenia się deterministycznych
automatów visibly pushdown~\cite{main}. Przedstawiono motywację do aktywnego uczenia się automatów.
Opisano algorytm wraz ze szczegółami implementacyjnymi oraz wykorzystanie więzów
w postaci systemu przepisywania słów, które usprawniają proces uczenia~\cite{srs}.
Następnie przedstawiono sposób generowania automatów oraz ich wykorzystanie do testowania algorytmu.
Na końcu przedstawione zostały wyniki eksperymentów.

}
\englishabstract{
The aim of this thesis is the implementation and performance analysis of an active learning algorithm
for deterministic visibly pushdown automata~\cite{main}. The motivation for active automata learning is presented.
The algorithm and its implementation details are described together with the use of constraints represented
as a string rewriting system, which improve the efficiency of the learning process~\cite{srs}.
Then, the method of generating automata and their use in testing the algorithm are presented.
Finally, the results of the experiments are presented.

}
