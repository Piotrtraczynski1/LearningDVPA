The already studied problem is how advice provided via an SRS reduces the number of equivalence queries 
necessary to learn an unknown language.
In this section addresses the converse: \emph{synthesis} of rewrite rules consistent with a given visibly pushdown language. 
Synthesized rewrite rules can serve as an explanation of the considered visibly pushdown language, capturing its key properties through concise rewrite rules.

There are two possible variants of that problem:
\begin{description}
\item[\textit{Synthesis from Automata}:] Given a \DVPA, generate string rewrite rules consistent with the language of the given \DVPA.
\item[\textit{Synthesis from Oracles}:] Given an oracle for an unknown visibly pushdown language, generate string rewrite rules consistent with the language. 
\end{description}
While (\textit{Synthesis from Oracles}) seems more natural, we show that even having a \DVPA the synthesis problem is challenging for two reasons: 
high computational complexity and an undetermined  objective (what is the right notion of a ``relevant rule''?). 
Therefore, (\textit{Synthesis from Oracles}) suffers from these same shortcomings, and further progress in (\textit{Synthesis from Automata}) is required before (\textit{Synthesis from Oracles}) can be effectively addressed.



We consider a \DVPA $\aut$ over a non-trivial alphabet (at least two letters). 
Furthermore, we assume that if the alphabet includes a call letter, it also includes a return letter, and vice versa.
The latter assumption is to ensure that there are other well-matched words than the empty word.


We establish a polynomial upper bound on the size of minimal-length rules consistent with $\aut$, which implies that such rules
can be computed in non-deterministic polynomial time.

In order to achieve that, we give necessary conditions for pairs of words to form a rewrite rule consistent with $\lang(\aut)$.
We consider first rules without variables, and then move to rules with a single variable, which is finally generalized to any bounded number
of variables.

\Paragraph{The size of a minimal consistent well-matched rule: the ground case}
Consider a well-matched word $w$. We associate with this word a function $f_w \colon Q \to Q$ transforming states into states as the congruence relation for regular languages.
Formally, for a state $q \in Q$ we define $f_w(q) = q'$ if and only if for some stack content $u$, we have $(q,u) \trans{\aut}{a} (q',u)$.
Note that ``some $u$'' can be changed to ``all $u$'' as the transitions over well-matched word do not depend on the initial stack content. 
If $w_1, w_2$ are well-matched word such that $f_{w_1} = f_{w_2}$, 
then for all words $x,y$ we have $x w_1 y \in \lang(\aut) \iff x w_2 y \in \lang(\aut)$~\cite[Theorem~1]{alur2005congruences}.
It follows that $w_1 \to w_2$ is a string rewrite rule consistent with $\lang(\aut)$. 
Setting $n = |Q|$, there are $n^n$ functions from $Q$ to $Q$, and hence there are $2^{\bigO(n \log n)}$ possible functions $f_w$, 
where $w$ is a well-matched words.
There are $2^\Theta(k)$ well-matched words of the length $k$, and hence over well balanced words of length $\bigO(n \log n)$ there is a pair of 
a well-matched words $\lhs, \rhs$ such that $\lhs \to \rhs$ is a ground well-matched rule consistent with $\lang(\aut)$. 

\Paragraph{The size of a minimal consistent well-matched rule: the general case}
For rules with variables, we consider only a subset of rules and only a sufficient condition for consistency. 
Still, we obtain asymptotically the same bound. 
Consider a rewrite rule with variables $\lhs \to \rhs$ of the form:
\begin{align*}
\lhs &= w_0 X_1 w_1 X_2 \ldots  X_k w_{k} \\
\rhs &= v_0 X_1 v_1 X_2 \ldots  X_k v_{k}
\end{align*}
where the order of variables in $\lhs$ and $\rhs$ is the same and all words $w_i,v_i$ all well-matched words over $\vAlph$ (do not contain variables).
Observe that if the corresponding words $w_i$ and $v_i$ define the same function, i.e., 
$f_{w_0} = f_{v_0}, \ldots, f_{w_{k}} = f_{v_k}$, then the rewrite rule $\lhs \to \rhs$ is consistent with the language of $\aut$. 
To see that, consider an instantiation of $\lhs \to \rhs$, i.e., 
consider a sequence of balanced word $b_1, \ldots, b_k$, which we substitute for $X_1, \ldots, X_k$
obtaining the ground rewrite rule:
\[
	w_0 b_1 w_1 b_2 \ldots  b_k w_{k} \to v_0 b_1 v_1 b_2 \ldots  b_k v_{k}
\]
Observe that the left hand side and the right hand side define the same function, and hence this ground rule is 
consistent with $\lang(\aut)$.
It follows that every rule from $\sem{\lhs \to \rhs}$ is consistent with $\lang(\aut)$, which by definition means that 
$\lhs \to \rhs$ is consistent with $\lang(\aut)$.
Now, observe that, setting $n = |Q|$, there are $n^{k\cdot n}$ different combinations of $k$ functions from $Q$ to $Q$. 
The number of well-balanced words is $2^{\Omega(n)}$ and hence there is a constant $c>0$ such that
among words of the form $w_0 X_1 w_1 X_2 \ldots  X_k w_{k}$ such that the sum of lengths of $w_0, \ldots, w_k$
 bounded by  $2^{c \cdot k\cdot n \cdot \log(n)}$, there is a pair of words, which forms
 a rewriting rule consistent with $\lang(\aut)$.


The counting argument shows that for a given $\DVPA$ $\aut$ there are short (of polynomial length in $|\aut|$) 
rewrite rules that are consistent with $\lang(\aut)$. 
Since testing whether a given rule (with bounded number of variables)
is consistent with $\lang(\aut)$ is decidable in polynomial time, we can find (some) rules consistent with $\lang(\aut)$ in 
polynomial time. Clearly, not all rules can be generated in non-deterministic polynomial time. 
Even for the \DFA case, we can construct an automaton $\aut$ such that there are words $w$ such that 
$w \to \epsilon$ is a rule consistent with $\lang(\aut)$, but every such $w$ is of exponential length. 
It suffices to define a DFA such that transitions over each letter is the same permutation of an exponential order.

While we have shown that rules can be generated in non-deterministic polynomial time, 
a natural question is whether this is optimal.

\Paragraph{Complexity of finding consistent rules}
We have established an upper bound on the minimal length of words defining the same function. 
We show that computing the exact minimal length of such words is difficult, which implies that
finding a minimal-length rewriting rule, which is consistent with a given \DVPA is difficult as well.
Furthermore, the problem is difficult already for \DFA.

We consider the \emph{minimal collision} problem defined as follows: decide, given a \DFA $\aut$ and $k>0$, whether 
there exists a pair of different words $w_1, w_2$ such that (1)~$|w_1|,|w_2| \leq k$, and
(2)~$f_{w_1} = f_{w_2}$. As we have discussed above for the ground rewrite rules, assuming that $\aut$ is over at least two-letter alphabet, 
for every $k > |\aut|\cdot \log(|\aut|)$ the answer is yes. 
Consequently, the problem is in \NP. We show that it is \NP-complete.

\begin{restatable}{lemma}{NPHardnessOfClash}
The minimal collision problem is \NP-complete.
\end{restatable}

The proof is by reduction from the 3-SAT problem.
The main difficulty of the proof is to ensure that if the formula is unsatisfiable, then no pair of short words define the same function from $Q$ to $Q$ .
We have constructed a gadget \DFA, such that unless one of the designed letters occurs in a word, each word of length at most $n$ generates the unique function.
We consider the alphabet consisting of literals from a given 3-CNF formula.
With that gadget \DFA, we enforce that short words having a collision, must contain a variable or its negation, i.e., represent an assignment,
and these word must contain a literal from each clause.

Another decision problem associated with finding consistent rules is deciding, for a given \DFA $\aut$ and $v \in \Sigma^*$, the minimal length of $w$ such that $f_w = f_v$.
We formalize this problem as follows. The \emph{bounded rewriting to single letter} problem asks,
given a \DFA $\aut$ over $\Sigma$,  a letter $a \in \Sigma$, and a bound $k>0$, whether there exists a word $w \in (\Sigma \setminus \set{a})^*$ such that 
(1)~$w \to a$ is a rule consistent with $\lang(\aut)$, and
(2)~$|w| \leq k$. 
This problem is closely related to the \emph{membership}  problem for monoids~\cite{Kozen77} and groups~\cite{DBLP:journals/tcs/Jerrum85}, which in turn reduce to the \emph{bounded automata intersection} problem~\cite{Kozen77,DBLP:conf/wia/Wareham00}, which 
asks, given $m,k>0$ and $m$ \DFA $\aut_1, \ldots, \aut_m$, whether 
there is $w$ such that $|w| \leq k$ and 
$w \in n\lang(\aut_i)$ for every $i$.
The bounded automata intersection problem is \PSPACE-complete if $k$ is given in binary~\cite{Kozen77} and \NP-complete if $k$ is given in unary~\cite{DBLP:conf/wia/Wareham00}.
The same complexity bounds apply to the bounded rewriting to single letter problem as there is a straightforward reduction from the bounded automata intersection problem:

\begin{lemma}
The bounded automata intersection problem with length parameter $k$ reduces in polynomial time to 
bounded rewriting to a single letter with the parameter $k$.
\end{lemma}

In summary, there is not algorithm for generating minimal-length consistent rules in deterministic polynomial time unless \PTIME = \NP.
Consequently, the further research on the rule synthesis problem should focus on efficient heuristics rather than exact algorithms.
